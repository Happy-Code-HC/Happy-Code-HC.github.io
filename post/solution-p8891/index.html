P8891 题解

· 2022-12-11 · # 题解

思路

根据异或的性质，可以得出有且仅有 x⊕x=0x\oplus x=0x⊕x=0。不同的两个数字异或结果一定不为 000。

因此对于每次操作，我们仅需将 ax←ax−ya_x \gets a_x-yax​←ax​−y 即可。当 x=0x=0x=0，我们什么都不做。

时间复杂度 O(n+m)O(n+m)O(n+m)，在 1≤n,m≤1061 \le n,m \le 10^61≤n,m≤106 情况下不会超时。

注意由于 −108≤y≤108-10^8 \le y \le 10^8−108≤y≤108，经过操作后 aaa 数组可能会超过 int 范围，需要使用 long long。

代码

#include<iostream>
using namespace std;
int n,m;
long long a[1000005];
int main(){
    cin>>n>>m;
    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    while(m--){
        int x,y;
        cin>>x>>y;
        if(x!=0){
            a[x]=a[x]-y;
        }
    }
    for(int i=1;i<=n;i++){
        cout<<a[i]<<' ';
    }
    return 0;
}
