P9471 题解

· 2023-07-23 · # 题解

很简单的签到题。

遍历字符串，判断每个字符是数字、小写还是大写，然后加到对应的统计变量里面。

#include<iostream>
using namespace std;
string s;
int ans1,ans2,ans3;
int main(){
	cin>>s;
	for(int i=0;i<s.length();i++){
		if(s[i]>='0' and s[i]<='9'){
			ans1++;
		}else if(s[i]>='a' and s[i]<='z'){
			ans2++;
		}else if(s[i]>='A' and s[i]<='Z'){
			ans3++;
		}
	}
	cout<<ans1<<' '<<ans2<<' '<<ans3<<endl;
	return 0;
}
