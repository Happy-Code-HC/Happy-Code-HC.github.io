P7901 题解

· 2021-10-21 · # 题解

思路

这题是道数学题。

每经过一次琢喵就会经过坐标为 (x,y)(x,y)(x,y) 的点，但是起点可以是的任意点，那么琢喵在哪里就对答案没有影响。

所以我们找矩形的点数 nnn 和步数 kkk 的关系,也就是找 2n×2n2n \times 2n2n×2n 和 kkk 的关系。每走一轮就会经过琢喵一遍，经过的次数是 k÷(2n×2n)k\div(2n \times 2n)k÷(2n×2n)。

提醒

xxx 和 yyy 在洛谷可以不读入，但在某些 OJ 就必须读入，否则会离奇报错，所以建议读入所有变量。
Python 做完除法要输出整数必须强制转换成 int 类型。
最后提醒一句：十年 OI 一场空，不开 long long 见祖宗。

#include<iostream>

int main(){
    cin>>n>>k>>x>>y;
    cout<<k/(n*2*n*2)<<endl;
    return 0;
}
